# ✦ LearnTrack — متابع التعلّم

## Full-Featured Learning Tracker PWA

### Features
- 📱 **PWA** — Install on phone, works offline
- 🌐 **Bilingual** — Arabic / English with instant toggle
- 🏷️ **Custom Categories** — Create your own with emoji + color
- 🖼️ **Cover Images** — Upload images for each item
- 📝 **Lessons/Chapters** — Add, check off, and take notes per unit
- ⭐ **Star Rating** — Rate items after completion
- 🔍 **Search & Filter** — Find anything instantly
- 📊 **Progress Dashboard** — Visual stats and tracking
- 💾 **Local Storage** — Data persists on your device
- 📤 **Export/Import** — Backup and restore your data as JSON
- ❤️ **Favorites** — Quick access to starred items
- 🔄 **Sort** — By date, progress, or name

---

## 🚀 Deploy Free on Vercel (3 minutes)

1. Go to **github.com** → New repository → name it `learntrack`
2. Upload ALL 5 files from this folder
3. Go to **vercel.com** → Sign in with GitHub → Import project → Deploy
4. Open the URL on your phone → Install as app

## 📱 Install on Phone
- **Android**: Chrome menu (⋮) → "Install app"
- **iPhone**: Safari share (□↑) → "Add to Home Screen"
